What i do

IT ENGINEER

As IT ENGINEER, We manage all it equipment from Desktops,Laptops, IP Phones, wifi routers/AP, and first level support for any software application required by the company. From Hardware to Software troubleshooting, New equipment Deployment and Testing, and help to improved the existing system using own skills in programing or using third-party software. First contact of support using ticketing tool.

INFRA ENGINEER

Data center monitoring and maintenance. From Servers to Network devices. Daily logs checking. Deployment of new equipment or system to server. Data safe keeping from databased, shared drive and email backup. Cabling and monitoring of network devices and coordinating with the vendor incase they are needed.

IT ENGINEER(OFFSHORE)

Seabased IT Infrustructure. From Data Center to Security system. Commisioning of Satellite system. Marine equipment deployment and security camera. All equipment must be durable for sea type vessel.
